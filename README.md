
# MicroTools — Text Formatter

A professional single-page web app to clean and transform text quickly.  
Perfect for removing extra spaces, changing case, cleaning text, and more.

---

## Features

### Whitespace
- **Trim** – Remove leading and trailing spaces.
- **Remove extra spaces** – Collapse multiple spaces and tabs into a single space.
- **Collapse multiple newlines** – Replace multiple consecutive newlines with a single newline.

### Case
- **UPPERCASE** – Convert all text to uppercase.
- **lowercase** – Convert all text to lowercase.
- **Title Case** – Capitalize the first letter of each word.
- **Sentence case** – Capitalize the first letter of each sentence.
- **snake_case** – Convert spaces to underscores and lowercase the text.
- **kebab-case** – Convert spaces to hyphens and lowercase the text.
- **Capitalize Each Word** – Capitalize the first letter of each word.

### Clean
- **Remove punctuation** – Strip out punctuation characters.
- **Strip non-ASCII** – Remove non-ASCII characters.
- **Remove empty lines** – Delete lines that are empty or contain only whitespace.

### Additional
- **Undo** – Revert the last change.
- **Clear** – Remove all text from the input.
